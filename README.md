# 🐾 PurrPaws

PurrPaws is a fullstack web application built using React and Express.

## 🚀 Tech Stack

Frontend:
- React (Vite)

Backend:
- Node.js
- Express

Database:
- MongoDB

---

## 📁 Project Structure

```
cbsweg1-purrpaws/
│
├── client/   # React frontend
├── server/   # Express REST API
```

---

## 🛠 How To Run

### 1️⃣ Run Backend

```
cd server
npm install
node index.js
```

### 2️⃣ Run Frontend

```
cd client
npm install
npm run dev
```
